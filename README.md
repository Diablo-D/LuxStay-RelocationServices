# Service Apartments – Relocation Website

A modern, elegant, and fully responsive landing page for premium serviced apartments and corporate relocation services.

Built from scratch with a clean dark-mode aesthetic, smooth micro-interactions, and a luxury-real-estate feel – perfect for high-end B2B and corporate housing providers in 2025.

## Features
- Sleek dark UI with electric blue accents  
- Full-screen parallax hero section  
- Smooth scroll-triggered fade-in animations  
- Responsive card carousel for popular locations  
- Modern form design with floating-focus effects  
- Sticky header with glass-morphism effect  
- Mobile-first layout (looks great on every device)  

## Tech Stack
- HTML5  
- CSS3 (Flexbox, Grid, custom properties, modern animations)  
- Google Fonts – Inter (clean & professional)  
- Unsplash images (placeholder – easy to swap)
